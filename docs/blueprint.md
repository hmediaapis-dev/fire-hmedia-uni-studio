# **App Name**: H Media Unitrack

## Core Features:

- Dashboard Overview: Interactive dashboard offering at-a-glance view of total/available units, active tenants and open balances.
- Tenant Management: Manage tenant data with options for creating, reading, updating, and deleting profiles; each containing contact details, unit lists, rates, notes, and balance information.
- Unit Management: Real-time view of unit availability, with tools for managing unit details such as size, rate, rental status, tenant assignment, and gate codes. Unit status displayed visually.
- Billing and Invoicing: Allows for creation of invoices, tracking payment status (paid/void), setting recurring billing, viewing tenant history and recording manual payment entries.
- Automated Billing: Allows for automated monthly invoice generation based on unit rental data via a scheduled function. It will include amounts, due dates, payment records, and status (voided, unit relation).
- Form Management: Enables secure storage and management of documents (PDF, DOCX). Features upload, download, edit and delete forms, each with associated metadata: title, description, fileType and upload date.
- Site Configuration: Configure application settings (gate codes, default unit rates, contact information, and invoice day), securely stored and managed.

## Style Guidelines:

- Primary color: HSL(210, 67%, 53%) - A medium-toned blue (#3498DB) evoking trust and security.
- Background color: HSL(210, 20%, 95%) - A very light desaturated blue (#F0F8FF), provides a clean and calming backdrop.
- Accent color: HSL(180, 50%, 50%) - A bright cyan (#46D0CF), for interactive elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look. It is suitable for both headlines and body text.
- Crisp, minimalist icons representing various functions such as user management, units, billing, and settings. Consistent line weights.
- Clean, card-based layout with intuitive sidebar navigation for accessing different sections. Responsive design to accommodate various screen sizes.
- Subtle transitions and animations on content loading and UI interactions to improve user experience without being distracting.
